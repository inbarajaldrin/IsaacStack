### **Play Simulation**

```python
import omni.timeline

timeline = omni.timeline.get_timeline_interface()
timeline.play()
print("Simulation started.")
```

---

### **Pause Simulation**

```python
import omni.timeline

timeline = omni.timeline.get_timeline_interface()
timeline.pause()
print("Simulation paused.")
```

---

### **Stop Simulation**

```python
import omni.timeline

timeline = omni.timeline.get_timeline_interface()
timeline.stop()
print("Simulation stopped and reset.")
```
